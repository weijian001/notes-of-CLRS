# 13.4 Deletion

标签： 红黑树删除

---
###笔记
与n个结点的红黑树上其他的基本操作一样，删除一个结点要花费O(lgn)时间。与插入操作相比，删除操作要稍微复杂些。

```c++
//红黑树的删除同样要用到Transplant，不过这里稍微有点不同
RBTransplant(T, u, v)
    if u.p == T.nil     //不同1
        T.root = v
    else if u == u.p.left
        u.p.left = v
    else
        u.p.right = v
    v.p = u.p           //不同2
```
